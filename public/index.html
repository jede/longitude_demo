What is it?

Longitude is a set of calculations for creating mobile first, responsive and semantic web sites. It is written purly in SCSS and is meant to be as unobtrusive as possible, meaning that you shouldn't have to adapt your semantics to the CSS.

Since Longitude only consits of calculations, it doesnt add any overhead to your CSS. No extra stylesheets to include in your final CSS, nothing.

What is it not?

Longitude is not a set of classes, it is a set of calculations. This means that Longitude itself is not a grid. You have to create the grid!

Fear not! Longitude makes it a breeze. It is acctually really handy to be able to taylor make the grid for exclusivly for the web site you are creating!

Longitude is not a collection of UI styles, it just helpes you with the responsive layout, thats all. So if you want to combine it with something else, go ahead! Or if you want to rol your own, please feel free. Longitude will stay out of your way.
